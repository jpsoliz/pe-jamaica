UNIT 1 – Introduction to ArcGIS Pro
Create a New Project
This exercise is to create a new project in ArcGIS Pro (3.7.x)
• Open ArcGIS Pro through the Windows Start Menu
o Go to Start > ArcGIS > ArcGIS Pro
• On the intro screen
o Go to the New Project area and select Map
• On the Create a New Project window
o Enter UNIT01 Intro for the Name
o Use the Browse folder to navigate to the C:\Training\UNIT01 folder to set the Location
o Leave ‘Create a folder for this project’ unchecked
Click OK
• The project is created in the folder you specified. Note that the C:\Training\UNIT01 directory
includes multiple files, but the important project file ends in .aprx and has a suitcase icon. In our
example, the project file will be UNIT01 Intro.aprx

Explore the Catalog Pane
The Catalog Pane is useful to customize your project and access items like maps, layouts, folders and
databases quickly. The following is to show some of the features of the Catalog Pane.
Opening the Catalog Pane
The Catalog Pane is  typically open by default when you create a new project. But sometimes, you will
have to open it manually:
• On the View ribbon, click on Catalog Pane (Windows group)
• The Catalog Pane will typically open on the right-hand side of your ArcGIS Pro project window:
Exploring the Catalog Pane - Maps
• Expand the Maps folder by clicking on the small triangle next to Maps in the Catalog Pane
• When it expands you will see an object called Map underneath it. This refers to the Map you see
in the middle of the ArcGIS Pro window.
• Close the Map in the ArcGIS Pro window by clicking the X on the tab at the top of the map

Changing Unit Properties
You may have noticed that be default, measurements of distance and area are set to two decimals, and
defaults to feet. You can change this by setting unit options for the project.
• Go to the Project tab at the top left corner of ArcGIS Pro
• In the sidebar, select Options
• A new window opens. At the top of this window’s sidebar, under the Project group, select Units
• Open the Distance Units and change the Default setting to Meter
• Click its Format box and change the number of decimals to 3, click OK
• The Distance Units should read as:
• You can do the same with the Area units. Here we changed the Default to Square Meter and set
both Square Meter and Hectare to 3 decimals
• Click OK when finished. Feel free to remeasure a few things to note the change in decimals for
the distance and area units

Symbolizing Data

anyone reading the map can recognize features by color, or at least note a contrast between features
rather quickly. This is called symbology in the GIS world, and we will make some changes to symbology in
our project. For this exercise, we will symbolize the data so that we assign a color to parcels that are
registered with title and another color for parcels that are not.
• Right-click on St. Andrew Parcels on the Contents pane and select Symbology from the menu

colors have already been assigned for the PARCEL_STATUS field. This is because the layer is
subtyped, meaning there is a built-in categorization of the parcels. The field used for the subtype
is symbolized by default
